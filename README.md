## Descrição do projeto 

<p align="justify">
O Jogo da Forca é um projeto desenvolvido como parte do curso de Ciência de Dados e Machine Learning, na disciplina Desenvolvimento para Ciências de Dados I. A aplicação foi construída em Python utilizando Tkinter, trazendo uma interface moderna em tons de lilás, rosa e azul para uma experiência visual agradável. Além da interface refinada, o projeto aplica princípios importantes da área, como modularização, organização de código e separação entre lógica e apresentação. O resultado é um jogo interativo, intuitivo e funcional, que demonstra habilidades essenciais para o desenvolvimento de aplicações dentro do ecossistema de ciência de dados.


![Descrição do projeto Jogo da forca, uma experiência visual leve e intuitiva para explorar lógica, interação e diversão no clássico Jogo da Forca.](./assets/hangman_illustration.png)
</p>

## Funcionalidades

:heavy_check_mark: `Funcionalidade 1:` Interface gráfica moderna com paleta personalizada, exibindo título, área de jogo, teclado virtual e mensagens dinâmicas.

:heavy_check_mark: `Funcionalidade 2:`Exibição da palavra oculta, atualizando automaticamente conforme o jogador acerta ou erra letras.

:heavy_check_mark: `Funcionalidade 3:` Desenho progressivo da forca e do boneco, evoluindo de acordo com o número de erros cometidos.

:heavy_check_mark: `Funcionalidade 4:` Teclado virtual interativo, desativando letras já utilizadas e oferecendo feedback visual.

:heavy_check_mark: `Funcionalidade 5:`Botão de “Novo Jogo”, reiniciando toda a interface e gerando automaticamente uma nova palavra.

:heavy_check_mark: `Funcionalidade 6:` Sistema de lógica separado da interface (arquitetura modular), facilitando manutenção, testes e expansão.

:heavy_check_mark: `Funcionalidade 7:`Suporte a listas personalizadas de palavras, permitindo temas específicos conforme a necessidade do projeto.

## Abrir e rodar o projeto

Após baixar o projeto, você pode executá-lo diretamente no seu ambiente Python. Para isso, siga os passos:

- `Certifique-se de ter o Python instalado em sua máquina (versão 3.8 ou superior).;
-Baixe o projeto e extraia os arquivos caso tenha vindo em formato .zip.
- Abra a pasta do projeto no editor de sua preferência (VS Code, PyCharm, etc.).
- No terminal, navegue até o diretório onde está o arquivo principal do jogo.
- Execute o comando:


python main.py
- A interface gráfica será aberta automaticamente, permitindo jogar o Jogo da Forca .
